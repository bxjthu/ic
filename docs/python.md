## 安装 Python 工作环境

* Python3

* Anaconda

[Ananconda](https://www.continuum.io)是一个包含了很多科学实验、工程设计、数据分析所需软件包的Python平台.  

Anaconda比较大, 建议只安装 miniconda, 通过[清华大学镜像](https://mirror.tuna.tsinghua.edu.cn/help/anaconda/) 安装需要的软件包(numpy, scipy, matplotlib等).
